## Usage

This PID contorller is used to control the speed of a nacell. This is done by setting different Kp and Kd values for this PD controller. Using the Comm port there are 5 options for tunning the PID controller. Each case is automatically activated by pressing the corresponding button 1-5 on the keyboard. 

* Case 1: Adds 0.05 to the Kp value.

* Case 2: Removes 0.05 from the Kp value. 

* Case 3: Adds 0.001 to the Kd value. 

* Case 4: Removes 0.001 from the Kd value. 

* Case 5: Displays the Current Kp and Kd values.
